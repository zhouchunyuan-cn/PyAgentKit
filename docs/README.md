# PyAgentKit 开发教程

这份教程详细讲解 PyAgentKit 每个模块的**设计原理与工作机制**，帮助你理解一个智能体（Agent）框架是如何从零搭建起来的，以及如何用它开发自己的 AI 应用。

## 这份教程适合谁

- 想理解 **ReAct 推理循环**、工具调用、多智能体协作原理的开发者
- 正在构建 AI Agent 应用，需要参考架构设计的工程师
- 对"大模型 + 工具 + 记忆 + 协作"这套范式感兴趣的学习者

## 两条学习路径

**路径 A：理解核心（最少阅读）**
只想懂 Agent 怎么工作 → 读 01、02、03 三章，理解 ReAct + 工具即可上手。

**路径 B：完整掌握（推荐）**
想用全部能力 → 按顺序读完 8 章，最后用第 8 章的 YAML 配置编排你自己的多 Agent 系统。

## 约定

- 所有代码示例都基于真实 API，可直接复制运行（需先配置 `ZHIPUAI_API_KEY`）
- `core/xxx.py` 的引用都指向项目源码，便于对照阅读
- 默认模型是 `glm-4-flash`（完全免费）
