# 第 1 章：快速开始

> 目标：把 PyAgentKit 跑起来，理解"一个能调用工具的 Agent"长什么样。

## 1.1 安装

```bash
git clone https://github.com/zhouchunyuan-cn/PyAgentKit.git
cd PyAgentKit
pip install -e .
```

`pip install -e .` 会做两件事：
1. 安装依赖（zhipuai、python-dotenv、pyyaml、scikit-learn、numpy 等）
2. 注册 `pyagentkit` 命令（基于 `pyproject.toml` 里的 `[project.scripts]`）

## 1.2 配置 API Key

PyAgentKit 用智谱 GLM 作为大模型后端。获取 Key：

1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 注册账号（赠送免费额度）
3. 在控制台获取 API Key

然后配置：

```bash
cp .env.example .env
# 编辑 .env，填入你的 Key
```

```
ZHIPUAI_API_KEY=你的Key
```

> 默认模型 `glm-4-flash` 是**完全免费**的，足够学习和原型开发。

## 1.3 命令行快速体验

最简单的方式——直接用命令行对话：

```bash
# 单次提问
pyagentkit ask "用三句话介绍量子计算"

# 交互式对话（带网络搜索和计算工具）
pyagentkit chat --tools web_search calculator

# 查看可用工具
pyagentkit tools
```

交互对话中输入 `:q` 退出，`:clear` 清空上下文。

## 1.4 第一个 Agent（代码版）

命令行适合快速体验。要构建更复杂的逻辑，需要用代码。下面是一个**会算数的 Agent**：

```python
from core import GLMClient, Agent, CalculatorTool

# 1. 创建 LLM 客户端（Key 自动从 .env 读取）
llm = GLMClient(model="glm-4-flash")

# 2. 定义 Agent
class MathAgent(Agent):
    def __init__(self):
        super().__init__(
            agent_id="math",
            name="数学助手",
            system_prompt="你是数学专家，需要计算时调用 calculator 工具。",
            llm_client=llm,
            capabilities=["calculate"],
        )
        # 3. 给 Agent 挂载工具
        self.tool_registry.register(CalculatorTool())

    def receive(self, message):
        # 收到消息时，调用 think() 进行推理
        reply = self.think(message.content)
        print(f"[{self.name}] {reply}")

# 4. 使用
agent = MathAgent()
print(agent.think("计算 (15 + 27) * 3"))   # 输出: 126
```

## 1.5 这里发生了什么

当你调用 `agent.think("计算 (15 + 27) * 3")` 时，Agent 内部发生了一个 **ReAct 循环**：

```
1. Agent 把"你是数学专家..." + 用户问题 组装成对话，发给 GLM
2. GLM 回复："我需要调用 calculator 工具，参数是 (15+27)*3"
3. Agent 执行 calculator 工具，得到结果 126
4. Agent 把工具结果回传给 GLM："calculator 返回 126"
5. GLM 回复："(15+27)*3 的结果是 126"
6. Agent 返回最终答复
```

这个"思考 → 调工具 → 观察 → 再思考"的循环，就是智能体的核心。下一章我们会深入讲解它的实现原理。

## 1.6 关键概念速览

| 概念 | 含义 | 对应模块 |
|------|------|----------|
| **Agent** | 能思考、用工具、有记忆的智能单元 | `core/agent.py` |
| **LLMClient** | 大模型客户端（封装 GLM 调用） | `core/llm.py` |
| **Tool** | Agent 可调用的外部能力（计算、搜索等） | `core/tools.py` |
| **think()** | Agent 的推理入口（ReAct 循环） | `core/agent.py` |
| **Memory** | Agent 的记忆（短期/长期） | `core/memory.py` |
